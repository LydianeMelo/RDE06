Resumo aula RDE03-Conceitos fundamentais e HTML

Tecnologias:

HTML5: Linguagem de marcação de hipertexto, responsável pelo conteúdo que aparece no Site. Como:

- Textos, Tabelas, Imagens e Vídeos.

É indicado através de tags, principalmente da < html> início e final que é com a barra </html>.

CSS3: Estilização do conteúdo, o design para o site ou página. Como:

- Cores, Fontes, Tamanho, Sombras, Posicionamento.

JS: JavaScript utilizado para a parte lógica, cálculos. Como:

- Interatividade, Menus interativos, validação, Popus animação.

Página Web e Website

Página Web ou Webpage: É um arquivo onde é inserido textos, imagens, tabelas e incorporar planilhas, apresentações, vídeos com ligações de hipertexto que permitem a navegação de uma página ou seção para outra.

Websites ou Sites: São uma coleção de páginas Web armazenadas em uma pasta ou em subpastas relacionadas de um servidor web.

Arquitetura Cliente-Servidor

Ocorre quando o cliente faz uma requisição ao servidor, um processamento de lado do servidor, ou seja, no Server-Side e retorna para o cliente.

Server-Side: É preciso ter tecnologias que sejam capazes de atender ao cliente sendo linguagens de programação, banco de dados, entre outros. Também é necessário ter desenvolvedores ou profissionais que entende das tecnologias de Server-Side feito o processamento, o servidor retorna para o Client-Side.

Client-Side: Tem tecnologias de lado de cliente que seria HTML, CSS, JS e a AJAX, além de outras.

- Profissinais: Front-End, que trabalham com a HTML, CSS, JS, do lado do Cliente ou Client-Side.

- Desenvolvedores: Back-End, trabalham do lado do servidor ou Server-Side.

- FullStack: É aqueles que atuam tanto como Front-End e Back-End, precisam de conhecimento em linguagem de programação, banco de dados, entre outros.

HTML-Hypertex Markup Language

É uma Linguagem de Marcação de Hipertexto, baseada em tags criada na década de 1990 por Tim Bernes- Lee, para publicação de conteúdo em hipertexto, tais como: texto, imagem, áudio e vídeo. E que podem ser vinculados através de hiperlinks.
