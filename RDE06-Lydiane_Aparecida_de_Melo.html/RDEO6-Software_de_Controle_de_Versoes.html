SOFTWARE DE CONTROLE DE VERSÕES

Quando desenvolvemos um projeto seja codificando um software ou site, documentos digitalizados como no word, entre outros.
Realizamos os seguintes passos:

  • Criamos uma pasta no computador;
  • Criamos um arquivo com esse projeto e nele desenvolvemos;
  • Por fim, depois de concluído com o projeto realizamos a entrega.

Como realiza o Controle de Versões:

Através do Software que vai criar um Repositório Local e vai criar a partir do arquivo existente, um histórico de versões. Então, esse arquivo será direcionado para versão 1, constando todas as alterações que foram finalizadas, vai estar a versão 2, versão 3, tudo no mesmo arquivo mantendo um histórico dessas versões.
Além do Repositório Local, pode também fazer um Repositório Remoto(principalmente em equipe) em que os seus arquivos, projetos podem estar na nuvem.

Como funciona com o Repositório Remoto:

Vai ter um servidor específico para trabalhar com seu repositório remoto e ele vai gerenciar também. Então, um Software de Controle de Versão, vai gerenciar tanto o histórico de versão local, quanto o histórico de versão remoto e ele vai fazer sempre essa sincronização para tudo que está no repositório local esteja igual ao que está no repositório remoto e vice-versa.
